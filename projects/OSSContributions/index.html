Open-Source Contributions

my contributions to open-source software

Apart from my own open-source projects, I have a few contributions to some other open-source repositories.

Some of my contributions:

  • Agda standard library: I formalized and proved basic theorems on if_then_else_ expressions, some lemmata on lists and in particular non-empty lists, a lemma for minus on natural numbers and subsequent refactorings, and I fixed a type error
  • Agda2HS: I extended the internal standard library of Agda within Agda2HS with definitions, theorems, and proofs on algebraic properties and numbers, including natural numbers, integers, and machine words. Some proofs where migrations from the real Agda standard library, some where new.
  • Programming Language Foundations in Agda (PLFA): I added a paragraph to the PLFA book clarifying some confusion on modeling propositions as types (i.e., Decidable types) versus good old Booleans.

Some minor contributions:

  • fix crash in emacs everforest theme
  • document proper setup of the catppuccin theme for doom emacs
